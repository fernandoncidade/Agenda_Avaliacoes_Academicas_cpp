Aviso de Licenças e Atribuições de Software de Terceiros
Projeto: AGENDA DE AVALIAÇÕES ACADÊMICAS
Versão 2026.4.24.0, 24 de abril de 2026

Este documento lista as principais bibliotecas e componentes de terceiros utilizados no projeto AGENDA DE AVALIAÇÕES ACADÊMICAS e fornece referências às suas licenças. O objetivo é cumprir requisitos de distribuição e oferecer informações claras para usuários, integradores e revisores. Os módulos principais do projeto e da build estão definidos no CMakeLists.txt e refletidos na distribuição entregue ao usuário final.

Observação geral
- O projeto principal é licenciado sob GNU LGPL v3 (veja LICENSE). Este NOTICE concentra componentes de terceiros, atribuições complementares e obrigações de distribuição.
- O produto final é distribuído como executável Windows nativo, compilado a partir do código C++/Qt por meio de CMake. Dependências de execução, bibliotecas Qt, plugins e runtimes aplicáveis acompanham a build conforme o compilador utilizado.
- Os textos completos das licenças e os avisos de copyright de terceiros são fornecidos no pacote de distribuição/instalador (arquivos LICENSE, LICENSES, NOTICE ou diretório equivalente). O código-fonte correspondente do projeto principal deve acompanhar a distribuição ou estar disponível no repositório/canal de código associado; para componentes de terceiros cobertos por LGPL, siga as instruções do fornecedor aplicável ou do respectivo texto de licença.

Declaração sobre transmissão de dados e privacidade
- O executável padrão do projeto não envia telemetria automática a servidores externos.
- Todos os dados do usuário (por exemplo, banco de dados SQLite local, arquivos JSON de preferências, listas personalizadas e exportações PDF) são armazenados localmente no diretório persistente do usuário.
- Quaisquer integrações opcionais que impliquem transmissão de dados serão documentadas e requererão consentimento explícito do usuário.
- Para detalhes sobre tratamento de dados pessoais e privacidade, consulte a Política de Privacidade incluída no pacote (assets/PRIVACY_POLICY/Privacy_Policy_pt_BR.txt).

Notas sobre substituição de bibliotecas (quando aplicável)
